## 2024-03-20 - Accessible Chat Input
**Learning:** Adding ARIA labels to icon-only buttons significantly improves screen reader experience without visual clutter.
**Action:** Always check icon-only buttons for aria-labels.

## 2024-03-21 - Code Block Actions
**Learning:** Wrapping `pre` elements in a relative container allows for accessible actions (like copy) without disrupting markdown flow.
**Action:** Use a wrapper div for `pre` custom renderers in react-markdown.
