## 2024-03-20 - Accessible Chat Input
**Learning:** Adding ARIA labels to icon-only buttons significantly improves screen reader experience without visual clutter.
**Action:** Always check icon-only buttons for aria-labels.

## 2024-03-24 - Copy Code Button
**Learning:** For developer tools, a "Copy Code" button in code blocks is a critical UX feature that reduces friction significantly.
**Action:** Ensure all code display components include copy functionality with visual feedback.
